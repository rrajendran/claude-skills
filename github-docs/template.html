# Quick Start

Prerequisites

  • ✓ Node.js 20+ / Python 3.11+
  • ✓ Docker (optional)

1. Clone & Install

git clone https://github.com/org/repo.git
cd repo
cp .env.example .env
💡 Tip
Edit .env and set the required variables before starting.

2. Run

npm install && npm run dev
# or
docker compose up --build
ℹ️ Note
Replace with the actual start commands found in the project.

# Configuration

Copy .env.example to .env and set the values below.

VariableDefaultDescription
PORT8000Server port
DATABASE_URLsqlite:///./app.dbDatabase connection string
SECRET_KEYrequiredSecret key for auth tokens
⚠️ Security
Always change SECRET_KEY before deploying to production.
